# Kedai Cinta

Website Kedai Cinta merupakan website promosi yang dibuat untuk memenuhi tugas UAS mata kuliah Website Publishing.

## Fitur Website

- Beranda
- Tentang Kami
- Menu Favorit
- Galeri
- Lokasi
- Kontak

## Teknologi

- HTML5
- CSS3
- JavaScript

## Tujuan

Website ini dibuat sebagai media informasi dan promosi Kedai Cinta agar pelanggan dapat melihat menu, galeri, lokasi, dan informasi kontak secara online.
