# Product Requirements Document (PRD)

## Nama Project

Website Kedai Cinta

## Latar Belakang

Kedai Cinta membutuhkan media promosi berbasis website agar pelanggan dapat memperoleh informasi mengenai menu, lokasi, dan suasana kedai dengan mudah.

## Tujuan

Membuat website sederhana yang menarik, responsif, dan mudah digunakan sebagai media promosi Kedai Cinta.

## Target Pengguna

- Mahasiswa
- Pelajar
- Masyarakat umum
- Pelanggan Kedai Cinta

## Fitur

- Beranda
- Tentang Kami
- Menu Favorit
- Galeri
- Lokasi Google Maps
- Kontak

## Teknologi

- HTML
- CSS
- JavaScript

## Platform

Website dapat diakses melalui browser desktop maupun perangkat mobile.
